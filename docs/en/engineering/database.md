# Database

## Purpose

Database development and administration.

## Tools

### PostgreSQL

VSCode extension for connecting to PostgreSQL databases while developing.

### SQLTools

General SQL editor and connection manager.

### SQLTools PostgreSQL Driver

Provides PostgreSQL support for SQLTools.

### DBeaver Community

Primary database administration tool.

Used for:

- Database exploration
- Schema management
- Data editing
- ER diagrams
- Query execution
- Import and export
