# MY-DEV-ENVIRONMENT

My personal development environment.

I am getting to a point in my career where I am evaluating and re-evaluating multiple aspects, such as documentation, preparation, tooling, etc. The goal for this repository is to backup my configurations and keep a public profile of what I use and how I use it. I'll try to update is as often as I can and even post some of the reasoning for each item.

Depending on the period in my company I am developing solutions for Windows/MacOS/Linux. For the past year I have been using a Macbook Pro M4 supplied by my company, and in my personal desktop machines and Homelab at home all run Linux distributions which I'll vary and experiment sometimes, but mostly I'll run Fedora KDE, it just works for most of the things right out of the box.

## Goals

- Keep workstation configuration version controlled.
- Document important setup decisions.
- Make new machine setup simple and repeatable.
- Keep configuration organized and easy to maintain.
